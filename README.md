# Banks-Marketing——课程项目
目前传统的银行营销方式已逐渐显露出颓势，实现精准营销成为银行业面临的重要问题。

------

## 1.解题思路

选取UCI银行营销数据集和某金融机构的营销数据集作为研究对象，并进行数据清洗、数据分析和特征工程。同时利用SMOTE过采样解决数据集不平衡问题。最后使用随机森林、KNN、SVM等多种机器学习算法，构建了一个精确且稳健的银行精准营销预测模型。具体思路可看代码及注释
